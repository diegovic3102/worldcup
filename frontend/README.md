# Vue Frontend Project

This project is a Vue.js application structured to demonstrate the use of components, TypeScript, and Vite as the build tool.

## Project Structure

```
frontend
├── src
│   ├── components
│   │   └── HelloWorld.vue
│   ├── App.vue
│   ├── main.ts
│   └── types
│       └── index.ts
├── public
│   └── index.html
├── package.json
├── tsconfig.json
├── vite.config.ts
└── README.md
```

## Setup Instructions

1. **Clone the repository:**
   ```
   git clone <repository-url>
   cd frontend
   ```

2. **Install dependencies:**
   ```
   npm install
   ```

3. **Run the development server:**
   ```
   npm run dev
   ```

4. **Open your browser:**
   Navigate to `http://localhost:3000` to see your application in action.

## Usage

- The main component is `App.vue`, which includes the `HelloWorld` component.
- You can modify the `HelloWorld.vue` component to customize the greeting message.
- Add additional components in the `src/components` directory as needed.
